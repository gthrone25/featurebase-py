============
Installation
============


Stable release
--------------

To install featurebase_python, run this command in your terminal:

.. code-block:: console

    $ pip install featurebase_python

This is the preferred method to install featurebase_python, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for featurebase_python can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/gthrone25/featurebase_python

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/gthrone25/featurebase_python/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/gthrone25/featurebase_python
.. _tarball: https://github.com/gthrone25/featurebase_python/tarball/master
